CSS Box Model Explained

Content Area

Box Model Layers

  • Content
  • Padding
  • Border
  • Margin

Components of the CSS box model

  • Content: The innermost area where an element's actual content resides (text, images, media). Its dimensions are controlled by the width and height CSS properties.
  • Padding: Transparent space that surrounds the content and sits inside the border. It creates visual space between the content and the element's edge.
  • Border: The line between padding and margin that provides a visual boundary. Its style, width, and color are customizable.
  • Margin: The outer transparent space outside the border that creates a buffer between this element and adjacent elements.

Standard vs. alternative box models

There are two box models that determine how an element's total size is calculated when padding and borders are applied:

  • Standard box model (content-box): The default. Width and height apply only to the content area; padding and border are added on top, increasing the element's final size.
  • Alternative box model (border-box): With box-sizing: border-box, width and height include content, padding, and border. The final size stays consistent while the content area shrinks to accommodate padding/border.

Why the box model is important

Understanding the box model helps you:

  • Control spacing between elements.
  • Avoid unexpected element size changes that can break layouts, especially in responsive design.
  • Use browser developer tools to inspect and debug element dimensions.
