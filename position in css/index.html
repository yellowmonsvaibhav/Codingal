Understanding CSS Positioning

How the different position values affect element layout and flow.

Relative

Relative parent
Absolute (inside relative)

position: relative; keeps the element in normal flow but allows offsetting its box. It also creates the containing block for absolutely positioned descendants.

Absolute

Absolute (viewport or nearest positioned ancestor)

position: absolute; removes the element from normal flow and positions it relative to the nearest positioned ancestor (or the initial containing block).

Static

Static (normal flow)

position: static; the default. Elements are laid out in the normal document flow and ignore top/right/bottom/left offsets.

Sticky

Sticky (sticks to top when you scroll)

position: sticky; behaves like static until the element crosses a threshold (top) then it becomes fixed within its nearest scroll container.
